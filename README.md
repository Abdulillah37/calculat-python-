# مشروع الحاسبة البسيطة (Python Calculator)

مشروع بايثون بسيط للمبتدئين يقوم بالعمليات الرياضية الأساسية: جمع، طرح، ضرب، وقسمة.

---

## المتطلبات

- Python 3.x مثبت على جهازك
- أي محرر نصوص أو IDE لتشغيل الكود (مثل VS Code أو PyCharm)

---

## كيفية التشغيل

1. حمّل الملف calculator.py من الريبو.
2. افتح Terminal أو CMD في نفس مجلد المشروع.
3. شغّل البرنامج بالأمر:
   ```bash
   python calculator.py
